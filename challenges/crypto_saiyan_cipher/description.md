# Saiyan Cipher Scroll

Vegeta has encoded his battle plans using an ancient Saiyan cipher technique.

Intelligence recovered the following ciphertext from a intercepted transmission:

```
Akxeo_ku_hksrk_ulpcvo_58z
```

Decode it to reveal the secret flag.

**Note:** The flag value (inner content) is the decoded plaintext. Submit it as `THA{decoded_text}`.

*The key is the name of the Saiyan home planet — all lowercase.*
